# Browser Debug Utilities

Diese Dateien sind ausführbare Hilfen für Browser-Debugging und gehören bewusst nicht in die offizielle Projektdokumentation.

## Inhalte
- `debug-modal.js`: Hilfen für Modal-/Detailansicht-Debugging
- `debug-scroll.js`: Hilfen für Scroll-/Autohide-Debugging

## Nutzung
1. Browser-DevTools öffnen.
2. Inhalt der gewünschten Datei kopieren.
3. In die Konsole einfügen und ausführen.
